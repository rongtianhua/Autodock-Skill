
## 修复 12：Cache 清除 API（P1）✅

**新增函数：**
- `clear_cache(confirm=True)` — 交互式/静默清除全部缓存，返回删除文件数和释放空间
- `get_cache_info()` — 查看缓存占用（文件数、大小、文件列表）

**用法：**
```python
from autodock import clear_cache, get_cache_info

info = get_cache_info()
print(f"Cached: {info['n_files']} files, {info['size_mb']:.1f} MB")

cleared = clear_cache(confirm=False)  # 静默清除（自动化脚本）
print(f"Freed: {cleared['size_mb']:.1f} MB")
```
