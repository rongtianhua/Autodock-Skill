## PLIP 集成到 autodock 技能（分子对接）

### 背景
将 PLIP（Protein-Ligand Interaction Profiler）集成到 `autodock` 技能，替换/增强原有的 RDKit 几何距离检测。

### 环境准备
- **autodock313** 环境（Python 3.13）：`pip install pybel`
- pybel 依赖 `pkg_resources`（setuptools 的一部分）
- Python 3.13 + 最新 setuptools 不兼容：`setuptools 82.0` 移除了 `pkg_resources`的 `ImpImporter`
- **解决**：`pip install "setuptools==75.8.1"`（最后一个兼容版本）
- pybel 3.1.0 + Python 3.13 兼容性通过测试

### PLIP 9 种标准相互作用类型
| PLIP 属性 | 标准化类型 | 配体原子字段 | 说明 |
|-----------|-----------|-------------|------|
| `hydrophobic_contacts` | Hydrophobic | `item.ligatom` | 直接 pybel Atom |
| `hbonds_pdon` | H-bond | `item.a`（protisdon=True 时）| 蛋白为供体，配体为受体 |
| `hbonds_ldon` | H-bond | `item.d`（protisdon=False 时）| 配体为供体，蛋白为受体 |
| `pistacking` | π-π | `item.ligandring.atoms[0]` | 芳环第1个原子 |
| `pication_paro/laro` | π-cation | `item.ring.atoms[0]`（protcharged=True）| 注意是 `ring` 不是 `ligandring` |
| `saltbridge_lneg/pneg` | Salt bridge | **无直接 pybel Atom** | 存的是 RingCenter（带 .center 坐标），返回 None |
| `halogen_bonds` | Halogen bond | `item.don.x` | 卤素原子（非 `ligatom`） |
| `water_bridges` | Water bridge | `item.d` | 配体原子（非 `ligatom`） |
| `metal_complexes` | Metal complex | `item.target.atom` | 金属配位原子 |

### 关键调试经验

#### 1. pybel 无法直接读 PDBQT ROOT/BRANCH 格式
- Vina/ADT 的 docked PDBQT 文件有 `ROOT/ENDROOT/BRANCH/ENDBRANCH` 标记，pybel 无法解析
- **解决**：字符串解析提取 ATOM 行，再用 pybel 读取

#### 2. H-bond 的配体原子判断（最重要 bug）
- PLIP 源码：`ligatom = don.d.OBAtom if not protisdon else acc.a.OBAtom`
- `protisdon=True`：蛋白是供体 → 配体是**受体** → `item.a`
- `protisdon=False`：配体是供体 → 蛋白是**受体** → `item.d`
- 原代码错误地认为 `protisdon=True` 时用 `item.d`（错误！）

#### 3. π-cation 用错字段
- 原代码：`item.ligandring.atoms[0]`（不存在！）
- 正确：`item.ring.atoms[0]`（PLIP 统一用 `ring` 字段）

#### 4. 盐桥无法获取单个配体原子
- PLIP 盐桥存储的是 `RingCenter` 对象（带 center 坐标，无 `.atoms`）
- 无法用坐标映射到单个配体原子 → 对盐桥返回 `None`

#### 5. `_build_complex_pdb_for_plip` 构建策略
- 用 pybel 读受体 PDBQT → 转 PDB（而不是直接用原始 ATOM 行）
- 配体用字符串解析提取 → pybel 转换 → 只取 ATOM 行
- 去掉受体 PDB 末尾的 `CONECT/MASTER/END`，拼接后加 `END`

### PLIP 坐标映射原理
- pybel 和 RDKit 对同一配体分子的坐标完全一致（已验证）
- 精度选择 0.01 Å，`round(x, 2)` 后匹配
- 19/19 原子全部映射成功
