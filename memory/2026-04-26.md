### 一、结构文件获取（`_structure_fetch.py`）

**PDB 下载（AlphaFold / PDB）**
- PDB IDs → 优先尝试 PDB 格式；含空格/特殊字符 → `urllib.parse.quote()` 编码
- AlphaFold model_v6（**v4 已过时，返回 404**）
- AlphaFold `model.cif` → 需用 `PDBxmmCIFFile` 解析，CIF writer 写入 `.pdb`

**PubChem 配体获取**
- 化合物名含空格 → URL 编码
- SMILES 字段名：优先 `CanonicalSMILES`，回退到 `SMILES`
- 从 PubChem 获取后用 RDKit 生成 3D 构象（`AddCoords3D`）

**常见 PDB 处理问题**
- 共价修饰残基（PJE/02J/010）：meeko 不识别 → 过滤掉 HETATM 行中的非标准残基
- 含非标准残基的 PDB → `Polymer.from_pdb_string()` + `PDBQTWriterLegacy.write_from_polymer()`

### 六、PyMOL 可视化（`_pymol_viz_config.py`）

**核心教训**
- **叠加层过多**是观感混乱主因：cartoon + lines + labels + sticks + spheres 同时出现
- **配色碎片化**：spectrum/rainbow/CPK 同时存在
- **Ligand 表现过重**：sticks + spheres + H coloring 全部叠加
- **标签过密**：所有口袋残基都 label，字体 0.4 → 糊成一团
- **ray 参数缺失**：`antialias=2` 无 `ray=1` → 图片质量差

**5 种场景预设**

- `pocket`（口袋特写）：口袋 lines + 配体 gold，黑背景

### 七、PLIP 集成（2026-04-26）

**环境**
- `autodock313` 环境（Python 3.13）
- `pip install pybel`（依赖 `pkg_resources`）
- **setuptools 75.8.1**（82.0 移除了 `ImpImporter`，不兼容 Python 3.13）

**PLIP 全部 11 种相互作用类型字段映射**

| PLIP 属性 | 标准化类型 | 配体原子字段 | 说明 |
|-----------|-----------|-------------|------|
| `hydrophobic_contacts` | Hydrophobic | `item.ligatom` | 直接 pybel Atom |
| `hbonds_pdon` | H-bond | `item.a`（protisdon=True 时）| 蛋白为供体，配体为受体 |
| `hbonds_ldon` | H-bond | `item.d`（protisdon=False 时）| 配体为供体，蛋白为受体 |
| `pistacking` | π-π | `item.ligandring.atoms[0]` | 芳环第1个原子 |
| `pication_paro/laro` | π-cation | `item.ring.atoms[0]`（protcharged=True）| **注意是 `ring` 不是 `ligandring`** |
| `saltbridge_lneg/pneg` | Salt bridge | **无直接 pybel Atom** | 存的是 RingCenter（带 .center 坐标），返回 None |
| `halogen_bonds` | Halogen bond | `item.don.x` | 卤素原子（非 `ligatom`）|
| `water_bridges` | Water bridge | `item.d` | 配体原子（非 `ligatom`）|
| `metal_complexes` | Metal complex | `item.target.atom` | 金属配位原子 |

**关键调试经验**

1. **pybel 无法读 PDBQT ROOT/BRANCH 格式** — Vina/ADT 的 docked PDBQT 有 `ROOT/ENDROOT/BRANCH/ENDBRANCH` 标记，pybel 无法解析 → **字符串解析提取 ATOM 行**

2. **H-bond 的 `protisdon` 判断**（最重要 bug） — PLIP 源码：`ligatom = don.d if not protisdon else acc.a` → protisdon=True 时配体是**受体**（`item.a`），protisdon=False 时配体是**供体**（`item.d`）

3. **π-cation 用错字段** — 原代码 `item.ligandring.atoms[0]`（不存在！），正确：`item.ring.atoms[0]`

4. **盐桥无 pybel Atom** — PLIP 盐桥存储 RingCenter 对象，无 `.atoms` 属性，坐标映射返回 None

5. **`_build_complex_pdb_for_plip` 构建策略** — pybel 读受体 PDBQT→PDB + 字符串解析提取配体 ATOM 行 → 拼接后加 END

**坐标映射**：pybel 和 RDKit 对同一配体坐标完全一致，精度 0.01 Å，19/19 原子全部映射成功
