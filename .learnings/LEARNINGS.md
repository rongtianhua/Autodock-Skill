## 事实性错误类
### 1. skill 安装位置说错
- **错误**：把 `china-stock-analysis` 等 skill 说成"全局 skill"，实际在工作区 `~/.openclaw/workspace/skills/`
- **正确**：全局 = `/opt/homebrew/lib/node_modules/openclaw/skills/`，工作区 = `~/.openclaw/workspace/skills/`
- **教训**：涉及文件/目录时**必须先 `ls` 或 `read` 实际查看**，不能凭记忆推断
- **规则**：今后涉及 OpenClaw 配置/技能路径的问题，100% 通过命令确认

### 2. Kimi K2.5 价格对比说反了
- **错误**：声称 OpenRouter 的 Kimi K2.5（$0.42/百万输入）比国内 API ¥2/百万 还便宜
- **正确**：OpenRouter ≈ ¥3.0/百万，比国内 ¥2/百万 贵 50%
- **教训**：价格/数值比较类回复，必须把数据查清、算清、**再**得出结论，不能随口推断
- **规则**：价格比较必须标注来源和换算汇率，先查后写

### 3. MiniMax 遗漏
- **错误**："国内主流 Coding Plan 套餐"汇总时遗漏了 MiniMax
- **正确**：MiniMax Starter ¥29/月（首月 ¥9.9），无每周限额；OpenRouter 有 `minimax-m2.5:free`
- **教训**：汇总表类回复要对照搜索结果逐条核实，不要凭"印象中重要才列出"

### 4. 初次 akshare 测试 K 线"成功"误判
- **错误**：说第一次测试 K 线成功，实际两次都报 ProxyError

- **教训**：看输出时要区分"正常输出"和"错误信息"，不要只看了一半下结论
